With SDN olny a centralized controller is needed to control multiple devices in a network. Directly program the network.

About

Nowadays, big traditional networks aren't flexible enough to deal with their success. Futhermore, it's difficult to trace issues throughout a network topology. The response to these issues envolve rapid alterations to the network, which are next to impossible and usually preventive instead of reactive. Technologies we will use:

  • OpenFlow & P4
  • Prometheus
  • Grafana
  • Docker

With our project, we hope to achieve real time monitorization of a network. Create a dashboard for multiple users with control over the network. Develop a reactive configuration that can:

  • Re-route packages
  • Impose traffic limitations
  • Permit topoly dynamism
